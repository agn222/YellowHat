## 🛠Project Setup

For the Web-App 1.Clone the GitHub repo.
```bash
git clone <url>
```
2.Enter the client directory. Install all the required dependencies.
```bash
  cd frontend
  npm install
  npm start
```

3.To start the backend server:4
```bash
  cd backend
  npm install
  npm start
```
